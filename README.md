# eduSpark
Education tool to help teachers keep students focused and enhance learning experiences for all students, including students with autism and ADHD.
## Core Features
- Sparkling Rail: animated cursor rail that you can adjust its length, color and turn it on or off using the provided control.
- Image Display: choose the images you want to display. also you can zoom and move between them easily. (you can use mouse or keyboard arrow keys.
- Draggable Shapes: choose a shape then Right click to start drawing a shape. Move the mouse to resize the shape. also you can color, move, resize and clear the shapes.
- Highlighter: Left-click to start highlighting. Move the mouse to draw lines or shapes. Use the highlighter settings to customize its color, size, and opacity and clear it.
